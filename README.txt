PRINCIPALES CAMBIOS:

1. Menú Principal: Ahora solo cambia al achicar la pantalla a 960, verificar el número de botones y palabras 	extensas.

2. Slider y título: Si no existe imagen simplemente no aparece y el titulo se acomoda, si existe una la inserta y si más pone el slider.

3. Clases para las imagenes internas y destacados.

Destacados:
	.opcionImg : 			151 x 164  
	.opcionImgSmall: 		100 x 84
	.opcionImgVertical:		169 x 242	


Internas (en el contenido del centro): 
	img sin clase: 400 x 268
	.imgBig: 750 x alto automático
	.imgVertical 284 x alto automático
	.imgLeft: Flota la imagen a la izquierda
	.imgRight: Flota la imagen a la derecha

	la combinación de las dos ultimas clases hace que las imagenes floten según sea necesario
	Ejemplo: <img class="imgVertical imgRight" src="img/conservacion/internas/1.jpg">


Textos:
	.txtColor: Texto de diferente color dependiendo la sección
